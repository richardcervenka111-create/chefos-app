Chat cost compounds within a conversation. Chef's Assistant currently resends the full message history with every new question (no trimming, no prompt caching) — so message 1 of a chat costs a fraction of a cent, but message 10 costs more, because Claude re-reads the whole conversation each time. For occasional questions this is negligible; for long back-and-forth sessions it adds up faster than the per-message cost above suggests. Worth revisiting (history trimming or caching) if this feature gets heavy daily use.
